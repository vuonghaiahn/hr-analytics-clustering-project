# HR Analytics Clustering Project

## Project
Final project for the Data Analysis with Python course.

## Topic
Clustering analysis on the HR Analytics Case Study dataset.

## Dataset
HR Analytics Case Study dataset

## Repository structure
- `data/raw/`: original dataset files
- `data/processed/`: cleaned and merged data
- `notebooks/`: main notebook
- `figures/`: charts and images
- `src/`: helper scripts
- `report/`: PDF report / LaTeX files
- `slides/`: presentation slides
